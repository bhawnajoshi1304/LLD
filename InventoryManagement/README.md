## Design Patterns

- Factory Pattern

## Running the Application

```bash
mvn clean compile exec:java -Dexec.mainClass="com.lld.inventoryManagement.Main"
```

## Demo/Manual Mode

The application supports two modes:
- **Demo Mode**: Automated run with predefined use cases
- **Manual Mode**: Interactive user input for custom operations
